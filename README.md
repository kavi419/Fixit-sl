## ğŸš€ Overview

**FixIt SL** bridges the gap between citizens and local authorities. It allows users to snap a photo of an issue (like a pothole or garbage pile), automatically captures the location, and reports it to a central dashboard. Authorities can then view these issues on an interactive map and mark them as resolved.

## âš™ï¸ How to Run Locally

Follow these steps to run **FixIt SL** on your machine.

### 1. Clone the Repo
```bash
git clone https://github.com/kavi419/Fixit-sl.git
cd Fixit-sl
```

### 2. Backend Setup
```bash
# Install dependencies
npm install

# Create .env file and add your keys (MongoDB, Cloudinary, etc.)
# Start the server
node server.js
```

### 3. Frontend Setup
```bash
cd client
npm install
npm run dev
```
Open http://localhost:5173 in your browser.
